# EYazawa

Yazawa will give you some advice. Choose your worry. Your worries will disappear.

## Installation

Add this line to your application's Gemfile:

```shell
gem 'e_yazawa'
```

And then execute:

```shell
$ bundle install
```

Or install it yourself as:

```shell
$ gem install e_yazawa
```
